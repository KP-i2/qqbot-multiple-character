🚀 快速上手

01 启动 Dashboard — 双击桌面快捷方式或运行 dashboard_silent.bat，浏览器打开 http://localhost:8501

02 启动 NapCatQQ Desktop — 从开始菜单或 C:\Program Files\NapCatQQ Desktop\ 启动，等待 QQ 自动登录

03 启动 NoneBot2 — 在本页「进程管理」点击「启动 NoneBot2」

04 确认连接 — 检查顶栏 WS 指示灯变绿，表示 NapCat 已通过 WebSocket 连接到 NoneBot2

05 开始聊天 — 私聊或在群里 @机器人

⚙ 进程操作

操作说明
启动 NoneBot2启动 NoneBot2 服务，等待端口 8080 就绪
重启 NoneBot2关闭并重新启动 NoneBot2（NapCatQQ Desktop 会自动重连）
关闭 NoneBot2停止 NoneBot2 服务

🎨 情绪表情系统

Bot 回复后自动分析对话情绪，概率性发送表情图片。表情文件位于 skill_communication/emoji/。

操作说明
新增情绪在 emoji/ 下新建文件夹（文件夹名即情绪类别），放入图片和 keywords.txt
添加关键词编辑对应文件夹的 keywords.txt，每行一个关键词，# 开头为注释
添加图片放入 jpg/png/gif/webp 格式图片，同情绪多张图随机选取
热加载给 Bot 发送 /reloademoji，无需重启即可使用新表情

触发概率 50%，同一用户 60 秒冷却。当前情绪类别：angry / biaoxian / chigua / ganga / haoqi / happy / joker / ota / refuse / sad（可自行新增任意类别）。

🔗 链接内容抓取

用户分享 URL 时，Bot 自动抓取页面内容并据此对话。

链接类型处理方式
B站视频通过 B站 API 获取标题、简介、标签、热评等完整视频信息
b23.tv 短链自动解析重定向，识别 B站视频后走专用 API
其他网页通用 HTML 抓取，提取正文内容（标题 + 段落）

QQ 小程序和分享卡片也会被自动解析为可读文本，Bot 可以理解并回复。

🔧 NapCatQQ Desktop 配置

位于 C:\ProgramData\NapCatQQ Desktop\runtime\config\bot.json，由 NapCatQQ Desktop 管理。关键配置项：

配置项说明当前值
websocketClientsWebSocket 客户端连接配置（连接到 NoneBot2）ws://127.0.0.1:8080/onebot/v11/ws
autoStart启动 NapCatQQ Desktop 时自动启动 Bottrue
offlineAutoRestartQQ 离线时自动重启true
autoRestartSchedule定期重启（防止 QQ 掉线）每 12 小时

修改 bot.json 后需在 NapCatQQ Desktop 界面重启 Bot 才能生效。

❓ 常见问题

! 设备重启后恢复服务 — ①双击桌面快捷方式启动 Dashboard → ②启动 NapCatQQ Desktop → ③在 Dashboard 点「启动 NoneBot2」 → ④等待 WS 指示灯变绿

? WS 指示灯不亮 — NapCatQQ Desktop 未运行或 WS Client 配置错误，检查 bot.json 中 websocketClients 的 url 是否为 ws://127.0.0.1:8080/onebot/v11/ws

? 机器人不回复 — 检查 NoneBot2 是否运行 + WS 是否已连接 + NapCatQQ Desktop 中 QQ 是否在线

? NapCatQQ Desktop 中 QQ 离线 — 在 NapCatQQ Desktop 界面重新登录或重启 Bot

? 技能修改没生效 — 点击「重载技能」

? 蒸馏速度慢 — DeepSeek API，30-60 秒属正常

? 微博抓取失败 — 检查 Cookie 是否有效

⚙ Bot 行为

调整机器人基础行为参数，保存后即时生效（无需重启）。

⚡ 流式输出

控制流式输出的断句策略。字符阈值越小，分段越频繁；等待秒数越短，响应越快但可能断句不自然。

💾 历史管理

对话历史的容量与持久化策略。过期历史会在后台定期清理。

⏱ 响应时间

控制等待提示的触发时机。秒数越小，提示出现越快。

蒸馏配置

旧版本会自动备份。
