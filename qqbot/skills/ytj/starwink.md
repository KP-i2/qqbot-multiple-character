## 基本档案

| 项目 | 内容 |
|------|------|
| **名称** | StarWinK（原称 StarWink少女队） |
| **所属事务所** | STARRY（中国地下偶像两大事务所之一） |
| **所在地** | 上海 |
| **成立日期** | 2021年4月1日 |
| **首演日期** | 2021年5月3日 |
| **风格定位** | **王道系**偶像（与另一大事务所 Ultima Idols 的摇滚路线形成对比） |

| **早期 catch phrase** | 「一心同体、少女队」 |

> 名称继承自日本同名偶像组合，原属株式会社KTY旗下偶像企划 **StarWink! Project**。2023年与日本KTY断连后正式更名为 **StarWinK**。

## 发展历程

- **2021年** — 4月成立，5月首演
- **2022年** — 7月广州车祸（三名成员入院），12月发布首支原创曲《First Summer》
- **2023年** — 与日本KTY断连，更名为「StarWinK」；成员开始使用昵称活动；陆续推出《ホシノカギ》《ライファ》《Star Believers》《Polaris》等原创曲
- **2024年** — 新体制披露（夜夜、小芋加入），**首次东京远征**，发布多首原创曲
- **2025年** — **史上首支参加 TIF（TOKYO IDOL FESTIVAL）的中国内地日系偶像团体**，第二次东京远征 🏆
- **2026年** — 香港参演「JOY∞STAGE × DEAR STAGE・ASIA IDOL FES」，2月新体制披露（晨仔、海城加入），5月五周年纪念演出

## 圈内定位

StarWinK 与 Ultima Idols 并称为中国地下偶像两大事务所旗下代表团体。与 Ultima Idols 的摇滚/帅气路线形成对比，StarWinK 走的是**王道系**路线——正统偶像风格、以元气和笑容为核心、注重粉丝互动和舞台感染力。在中国地下偶像（地偶）圈中具有重要影响力。
