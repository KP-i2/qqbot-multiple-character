# 地下偶像常用用语与现场准则

## 前言

地下偶像（简称“地偶”），指不以主流媒体曝光为核心，主要通过小型剧场、Livehouse等线下演出现场进行活动的偶像团体或个人。中国的地下偶像文化主要借鉴自日本，同时又结合了本土特色，形成了独特的粉丝交流体系。本指南旨在帮助新人更好地理解地偶圈内的常用话语和现场约定俗成的行为规范。

## 一、常用缩写与简称

| 用语    | 含义               | 补充说明                                             |
| ----- | ---------------- | ------------------------------------------------ |
| 地偶    | 地下偶像的简称          | 最常用的指代称呼                                         |
| xox   | 小偶像              | 源自“偶像”拼音首字母XOX的组合，代指被喜欢的成员                       |
| ota   | 偶像宅              | 日文“御宅族（おたく, otaku）”的缩写，泛指粉丝群体                    |
| 推 / 担 | 喜欢的小偶像           | 即自己支持的对象；如“我的推是XX”“我推那个成员”                       |
| stf   | 工作人员             | 英文staff的缩写                                       |
| dm    | 私信               | 即“direct message”；简介中写“dm禁止”意味着不接受私信             |
| 切     | 拍立得照片            | 日文“チェキ（cheki）”的音译，指演出后与偶像拍摄的拍立得照片                |
| 拍切    | 和小偶像一起拍一张拍立得     | 特典会的主要活动之一                                       |
| 团切    | 和整个偶像团体一起拍集体拍立得  | 与全部成员一同合影的活动                                     |
| 宿题切   | 有主题的拍立得          | 按特定主题（如节日、服装主题）拍摄的切                              |
| 电切    | 线上视频通话拍立得        | 通过远程视频进行的一对一互动                                   |
| 远征    | 前往小偶像常驻地以外的地方应援  | 粉丝为了追特定的演出或活动前往外地                                |
| 起飞    | 演出时将观众托举起来       | 一种比较激烈的应援方式，视场地和团体情况有所限制                         |
| 噶比    | 男友粉 / 女友粉        | 将小偶像视作恋爱对象的粉丝类型                                  |
| 前世粉   | 从小偶像未出道时就开始追的粉丝  | 出道前就关注并支持对方的粉丝                                   |
| 亲友粉   | 对小偶像以朋友方式相处的粉丝   | 更偏向平等的朋友心态                                       |
| 无料    | 免费               | 日文“無料（むりょう）”的直接借用                                |
| 物料    | 周边               | 指各种粉丝商品如徽章、立牌、T恤等                                |
| 厄介    | 给周围观众造成麻烦的极端应援行为 | 包括过于激烈的动作、不顾他人安全的打call方式等                        |
| 同担拒否  | 拒绝与别人分享同一推的粉丝心态  | 源自日语“同担拒否（どうたんきょひ）”，指只能接受自己一个人推该偶像               |
| OOC   | 脱离人物原本设定         | 英文Out Of Character的缩写，多指同人创作中角色走形，也会用在偶像状态不符常规之时 |
| 地元    | 小偶像或团体的常驻活动城市    | 日文“地元（じもと）”对音借用，指日常演出的所在地                        |

## 二、应援相关用语

### 基础应援概念

| 用语          | 含义                   | 补充说明                                                                            |
| ----------- | -------------------- | ------------------------------------------------------------------------------- |
| 打 call      | 日系现场观众根据音乐节奏为偶像助威的行为 | 包括挥动荧光棒、喊口号、做固定动作等，是互动的基础方式                                                     |
| WOTA 艺（宅艺）  | 由观众表演的荧光棒舞蹈或成套应援动作   | 广义包括喊口号、MIX、PPPH等；狭义则特指用荧光棒进行独立舞蹈                                               |
| Mix（拉 mix）  | 具有固定节奏的长应援口号         | 最基本的Mix格式包含“Taiga, Faiya, Saiba, Daiba, Baiba, Jyaja”等连续音节；有更具风格的版本，在不同团体版本略有不同 |
| 振りコピ        | 简单模仿台上偶像的舞蹈动作        | 观众局部复制舞蹈动作，形成台上与台下的一体感                                                          |
| PPPH        | 现场用拍手加呼喊的经典节奏        | 多用于歌曲前奏或尾奏收束气氛                                                                  |
| ロミオ（罗密欧）    | 另一种固定应援节奏模式          | 名称源自该应援的特色节拍感                                                                   |
| 恋口上 / ガチ恋口上 | 表达“认真喜欢”的口上应援台词      | 通常在歌曲间奏或尾奏处齐喊，表现很深的应援心意                                                         |
| 巧克力口上       | 另一种常见应援口上            | 与恋口上齐名，被用来活跃气氛                                                                  |
| 饭撒（ファンサ）    | 针对粉丝做出的特别回应          | 如对镜头眨眼、飞吻、比心、招手、握手、击掌等                                                          |
| 虎！火！        | 日本Mix结构中常见的前缀音节      | 实际上是Mix的基础内容之一，读音有力生动                                                           |
| Oi！Oi！      | 间奏时齐呼的助兴口号           | 来自日系应援典型方式，用以凝聚场感                                                               |

### Mix 的基本结构示例

Mix是一种日本偶像文化中的长应援口号。基本的经典Mix通常包含以下音节（具体版本因团体和区域会有变化）：

Taiga, Faiya, Saiba, Daiba, Baiba, Jyaja ／ 虎！火！...

## 三、特典会相关用语

| 用语      | 含义            | 补充说明                                     |
| ------- | ------------- | ---------------------------------------- |
| 特典会     | 演出前后的互动活动     | 与偶像近距离聊天的环节，包括拍切、签名、简单对话等                |
| 特典券     | 进入特典会的凭证      | 通常按张出售，一张对应一次简短的交流或一次拍切；价格一般在几十至一百多人民币不等 |
| no shot | 特典时间内不进行拍照只聊天 | 可以更充分地利用时间进行深入对话                         |
| 大券      | 一次性购入多张特典券    | 能获得更长的互动时间，或用于多成员分开应援                    |
| 队列      | 排队的列次队列       | 若某位偶像比较火，特典会排队通常较长，寻找队尾牌开始等候             |
| 推し変     | 改变推的对象        | 放弃原推，改推另一位成员，有时是受演出状态或个人偏好的影响            |
| 1 shot  | 单人照片          | 只拍小偶像的拍立得照片                              |
| 2 shot  | 双人合影          | 粉丝和偶像同时出镜的照片 |
